Home-01：主页布局示意
  • 展示 macOS 窗口三件套结构：左侧 app-sidebar + 顶部 win-chrome-bar + 内容区 app-main
  • 展示 app-sidebar，含品牌区、「主要功能」分组（5 项）、「收藏夹」分组（3 项），当前页「首页」高亮
  • 展示 win-chrome-bar，含拖动区 + 帮助按钮 + 积分 chip + 用户头像 + 菜单按钮
  • 展示页头「Dashboard」标题（display 字体）+ 副文案 + 「新建项目」主色按钮
  • 展示 3 张统计卡片（grid-3）：活跃项目、待处理任务、本月用量，含数字、变化趋势、进度条
  • 展示「最近活动」列表（info-card），每行含品牌图标、标题、时间、状态 badge
  • 状态 badge 三色：成功（绿）、已同步（蓝）、待处理（黄）

Home-02：新建项目弹窗叠加态
  • 底层为 Home-01 失活态（暗化 + 点击穿透禁用）
  • 展示半透明遮罩覆盖主区
  • 展示居中弹窗 form-dialog，右上角有「X」关闭按钮（modal-close-x）
  • 展示弹窗标题「新建项目」
  • 展示三个表单字段：项目名称（input）、描述（textarea）、类型（combobox 下拉）
  • 展示底部操作区：「取消」（ghost）+ 「创建」（主色实心）按钮
  • 点击遮罩、X 或取消按钮关闭弹窗，回到 Home-01
