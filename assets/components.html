proto-gen 组件库

通用组件原语清单。写需求 / PRD 时直接引用类名（如「展示主按钮，应用 .btn-primary 风格」）， 不要在 PRD 文档中复述颜色 / 字号 / 圆角等具体值。视觉规范由 theme.css 注入，可一键切换主题。

仅展示通用组件原语；项目业务衍生类 （如 .channel-card / .agent-tabs / .apply-popover / .badge-free 等） 请查 references/css-components.md「业务衍生」段。
